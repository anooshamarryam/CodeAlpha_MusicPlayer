# Music Player

A simple and responsive Music Player built as part of my Frontend Development Internship task.

## Features
- Play, pause, and resume music
- Next and previous track controls
- Progress bar with playback tracking
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your web browser.

## Project Purpose
This project was developed to demonstrate fundamental frontend development skills, including responsive UI design, JavaScript audio controls, and DOM manipulation.
